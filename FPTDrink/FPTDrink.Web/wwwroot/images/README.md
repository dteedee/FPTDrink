# Thư mục Images

Đặt tất cả hình ảnh của website vào đây.

## Cấu trúc đề xuất:

```
images/
├── products/          # Hình ảnh sản phẩm
├── banners/          # Banner quảng cáo, carousel
├── categories/       # Hình ảnh danh mục
├── icons/            # Icon, logo nhỏ
├── placeholders/     # Ảnh placeholder mặc định
└── uploads/          # Ảnh người dùng upload (nếu có)
```

## Lưu ý:
- Tên file nên dùng chữ thường, không dấu, dùng dấu gạch ngang (-)
- Ví dụ: `tra-sua-truyen-thong.jpg` thay vì `Trà Sữa Truyền Thống.jpg`
- Nên tối ưu kích thước ảnh trước khi upload (khuyến nghị < 500KB)
- Format: JPG cho ảnh thật, PNG cho ảnh có nền trong suốt, WebP cho tối ưu nhất
